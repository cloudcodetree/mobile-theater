## Pelican 1510 (Carry-On Size)
```
External: 22.0 × 13.8 × 9.0" (559×351×229mm)
Internal: 19.8 × 11.0 × 7.6" (502×279×193mm)
Weight: 13 lbs empty

Best for: Command module (fits carry-on!)
```

## Pelican 1620 (Large)
```
External: 24.6 × 19.6 × 14.0" (625×500×356mm)
Internal: 21.4 × 16.4 × 12.5" (546×417×318mm)
Weight: 22 lbs empty

Best for: Multiple speaker pods
```

## Selection Criteria
- [ ] Waterproof (IP67 rating)
- [ ] Rolling wheels (for large cases)
- [ ] TSA-approved latches
- [ ] Pressure equalization valve
- [ ] Foam included or available

## Ordering
1. Measure all components carefully
2. Add 1" margin on all sides for foam
3. Check reviews for specific model
4. Consider buying foam separately (Kaizen)
