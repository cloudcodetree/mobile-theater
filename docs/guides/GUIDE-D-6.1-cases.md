# D-6.1 Pelican Case Selection

## Overview
**Time:** 2-3 hours | **Cost:** ~$200 (first case) | **Skill:** Beginner

## Case Requirements
| Case | Contents | Recommended Model |
|------|----------|-------------------|
| Command | Pi, switch, projector | Pelican 1510 |
| Speakers A | FL, FR, C, SL | Pelican 1620 |
| Speakers B | SR, RL, RR, stands | Pelican 1620 |
| Subwoofer | Sub + accessories | Soft case or 1650 |

## Pelican Case Specs
| Model | Internal (in) | Weight | Price |
|-------|---------------|--------|-------|
| 1510 | 20×11×7 | 11 lbs | $180 |
| 1620 | 22×17×10 | 18 lbs | $280 |
| 1650 | 29×18×10 | 23 lbs | $350 |

## Alternatives
- **SKB Cases:** Similar quality, sometimes cheaper
- **Nanuk:** Good mid-range option
- **Apache (Harbor Freight):** Budget ~$80-150

## Selection Checklist
- [ ] Measure all components
- [ ] Verify internal dimensions fit
- [ ] Check weight capacity
- [ ] Consider rolling vs carry
- [ ] Order from authorized dealer
