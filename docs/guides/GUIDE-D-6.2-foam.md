# D-6.2 Custom Foam Inserts

## Overview
**Time:** 4-6 hours | **Cost:** $120 | **Skill:** Beginner-Intermediate

## Foam Options
| Type | Pros | Cons | Cost |
|------|------|------|------|
| Pick-N-Pluck | Easy, included | Rough edges | Free |
| Kaizen | Clean cuts, layers | More work | $40/sheet |
| Custom CNC | Perfect fit | Expensive | $100+ |

## Tools Needed
- Electric knife or hot wire cutter
- Metal ruler
- Marker
- Templates (cardboard)

## Kaizen Foam Method

### Step 1: Make Templates
```
1. Place each item on cardboard
2. Trace outline with 5mm margin
3. Cut cardboard templates
4. Verify fit in case
```

### Step 2: Layer Planning
```
Pelican 1620 depth: 10 inches

Layer 1 (bottom): 2" - Base protection
Layer 2: 4" - Speaker pods
Layer 3: 2" - Chargers/cables  
Layer 4 (top): 2" - Lid padding
```

### Step 3: Cut Foam
```
1. Stack Kaizen sheets to desired thickness
2. Place template on top
3. Cut with electric knife (or exacto for thin)
4. Test fit components
5. Glue layers together (spray adhesive)
```

## Charging Access
Cut channels for charging cables:
```
   ┌───────┐
   │ POD   │
   │   ┌───┼─── Cable channel
   │   │   │    (cut through foam)
   └───┴───┘
```
