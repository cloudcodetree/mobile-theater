# D-6.3 Speaker Stands & Mounting

## Overview
**Time:** 2-3 hours | **Cost:** $110 | **Skill:** Beginner

## Stand Requirements
- Support 3kg (7 lbs) per pod
- Height: 1-2m adjustable
- Quick setup/teardown
- Fits in transport case

## Recommended Stands
| Type | Model | Height | Price |
|------|-------|--------|-------|
| Light stand | Amazon Basics | 7ft | $25/pair |
| Speaker stand | On-Stage SS7761B | 6ft | $60/pair |
| Tripod | Pyle PSTND2 | 6ft | $40/pair |

## Quantity Needed
| Location | Stands | Pods |
|----------|--------|------|
| Front (FL, C, FR) | 2-3 | Can share or use shelf |
| Surround (SL, SR) | 2 | On stands |
| Rear (RL, RR) | 2 | On stands |
| Sub | 0 | Floor placement |
| **Total** | **4-6** | |

## Universal Mount Bracket

### Option A: 3D Printed
```
┌─────────────────┐
│   SPEAKER POD   │
└────────┬────────┘
    ┌────┴────┐
    │ 1/4-20  │  ← Threaded insert
    │  mount  │     in pod bottom
    └────┬────┘
    ┌────┴────┐
    │ Adapter │  ← 3D printed
    │ 35mm    │     adapts 1/4-20 to
    │ socket  │     35mm pole
    └────┬────┘
         │
    [Stand Pole]
```

### Option B: Commercial
- Tilt bracket with 35mm adapter
- ~$10-15 each from Amazon

## Quick Deploy Setup
```
1. Extend stand legs (10 sec)
2. Raise pole to height (5 sec)
3. Attach pod to mount (5 sec)
4. Aim toward listening area (5 sec)

Total per stand: ~25 seconds
All 6 stands: ~3 minutes
```
